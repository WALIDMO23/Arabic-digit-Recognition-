# Arabic Handwritten Digit Recognition System

A complete system for recognizing Arabic handwritten digits (0-9) using Convolutional Neural Networks (CNN) with a user-friendly PyQt5 GUI.

## Features

- **Interactive Drawing Canvas**: Draw Arabic digits with your mouse
- **Real-time Recognition**: Instant digit recognition with confidence scores
- **Probability Visualization**: See probabilities for all digits (0-9)
- **Image Loading**: Load and recognize digits from image files
- **Model Training**: Complete training pipeline with data augmentation
- **Performance Metrics**: Detailed accuracy reports and training history
